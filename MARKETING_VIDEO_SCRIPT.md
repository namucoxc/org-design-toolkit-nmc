# MARKETING_VIDEO_SCRIPT.md — 推广短视频脚本

> 区别于 `DEMO_SCRIPT.md`（2-3 分钟产品技术演示），本文档面向**社交媒体推广**。包含 3 个时长版本（15s/30s/60s）和 4 个平台分发版本（小红书 / 抖音 / 视频号 / LinkedIn）。
>
> 核心目标：**5 秒留住人，15 秒制造好奇，30 秒激发行动**。

---

## 一、推广视频底层叙事框架

所有版本共用一套底层叙事：

```
痛点（你也是这样吗？）→ 反差（其实可以这样）→ 证明（看截图/数据）→ 行动（去 GitHub 找）
```

### 用户视角"痛点 vs 反差"对照

| 痛点（人人都懂） | 反差（颠覆认知） |
|---|---|
| 一份分工矩阵填 3 小时 | 5 分钟出图 |
| 改一稿就要重做 | 自动差异报告 v1→v2→v3 |
| 老板说"这不像我们公司"| 自动对齐你的官方案例 |
| 咨询师走了，矩阵就废了 | 模板 + 流程标准化沉淀 |
| RACI 看不懂，部门长不签字 | 中文"主责/参与/监督"+ 颜色编码 |

## 三、30 秒标准版（适合 视频号/B站/LinkedIn）

### 镜头脚本

| 时间 | 画面 | 字幕 | 旁白（女声平稳）|
|---|---|---|---|
| 0-3s | 黑屏淡入·白色大字 | "组织咨询师的一周" | "组织咨询师的一周，是这样的——" |
| 3-8s | 快剪：① 客户访谈 ② 翻看调研报告 ③ 打开 Excel | "诊断 1 天 → 方案 3 天" | "诊断 1 天，方案 3 天，其中——" |
| 8-13s | 屏幕特写·Excel 里手工填 RACI（加速 4x） | "10 部门 × 80 项" | "10 部门 80 项 RACI，手工填 2 到 4 小时。" |
| 13-15s | 镜头一转·咨询师揉眼睛 | "改一稿 = 重做一遍" | "老板一句『再改改』，重来。" |
| 15-20s | **画面翻转**·展示 8 份产物文件夹 | "现在 ↓" | "现在——" |
| 20-25s | 极速演示 10 轮对话 → 8 份文件出现 | "10 轮对话 · 5 分钟 · 8 份交付件" | "10 轮对话，5 分钟，输出 8 份完整交付件。" |
| 25-28s | GitHub 仓库截图 + 安装命令 | `claude plugins install org-design-toolkit` | "MIT 开源，欢迎贡献子行业模板。" |
| 28-30s | LOGO + slogan | "AI 是副驾驶，咨询师是决策者" | "AI 是副驾驶，咨询师是决策者。" |

### 视觉风格

- **前 15 秒**：低饱和度 / 灰调 / 慢镜头（衬托痛点）
- **后 15 秒**：高饱和度 / 蓝调 / 快镜头（衬托方案）
- **节奏对比**强烈

### 字幕风格

- 痛点字幕：宋体 / 灰白
- 方案字幕：思源黑体 Bold / 蓝色 `#2F5597`
- 数字必加 **粗体 + 高亮框**

## 四、60 秒完整版（适合 B站/油管/知乎视频）

### 6 段式结构

```
0-5s    钩子    : 一个真实的咨询师疲惫镜头
5-15s   痛点    : 3 个咨询师都懂的痛点
15-25s  转折    : 提出方案
25-45s  方案    : 演示 10 轮对话 → 8 份产物
45-55s  特性    : 10 项核心特性快闪
55-60s  CTA     : 安装命令 + slogan
```

### 6 段详细脚本

#### 段 1（0-5s）· 钩子

**画面**：深夜办公室·咨询师面前堆满文件夹·Excel 屏幕反光在眼镜上
**字幕**：(无)
**旁白**：(BGM 起·低调电子)

#### 段 2（5-15s）· 三个共鸣痛点

**画面**：快剪三组场景

| 场景 | 字幕 | 旁白 |
|---|---|---|
| 咨询师在白板上画组织架构 | "还在凭经验画矩阵？" | "你还在凭经验画矩阵——" |
| 客户说"这格式不像我们公司" | "客户说'不像我们的格式'" | "客户说不像他们公司的格式——" |
| 三个版本来回改 | "v1 v2 v3，每次重做" | "改 3 版，每次重做一遍？" |

#### 段 3（15-25s）· 转折

**画面**：屏幕渐变·从灰到亮
**字幕**：**这一切，从今天起结束。**
**旁白**：
> "其实，组织设计的格式劳动，本来就不应该咨询师来扛。"

#### 段 4（25-45s）· 演示

**画面**：屏幕录制·快剪 + 关键帧停顿

| 子镜头 | 时间 | 字幕 |
|---|---|---|
| 输入触发词 | 25-27s | "做一份示例企业的分工矩阵" |
| 自动 RAG 关联案例 | 27-30s | "✓ 自动 RAG 你的官方案例" |
| 10 轮对话快闪 | 30-37s | "10 轮对话采集（25-45 分钟）" |
| 8 份文件出现 | 37-42s | "✓ 8 份交付件 · 5 分钟生成" |
| 双击打开 .xlsx | 42-45s | "RACI 中文化 + 产出表单完整" |

#### 段 5（45-55s）· 10 项特性快闪

每项 1 秒，配 icon + 关键词：

```
🔍 RAG 强制前置
📊 三张矩阵分表
🇨🇳 RACI 中文本地化
📋 产出表单必填
🔄 流程驱动排序
🔗 表间承上启下
🏭 7 个子行业模板
⚡ 三个专项动作
📈 增量迭代机制
✅ 七道质量闸
```

**旁白**：
> "10 项特性，每一项都来自真实项目的踩坑。"

#### 段 6（55-60s）· CTA

**画面**：
- 全屏 GitHub 仓库主页截图（高亮 star 按钮）
- 底部安装命令

```bash
claude plugins install org-design-toolkit
```

**字幕**：
```
立即使用 · MIT 开源 · 期待你的子行业模板贡献
```

**旁白**：
> "GitHub 搜索 org-design-toolkit。AI 是副驾驶，咨询师是决策者。"

## 六、视频制作 Checklist

### 录制前

- [ ] 准备脱敏的"示例企业"数据
- [ ] 确认 Claude Code 安装并测试触发流程顺畅
- [ ] 在干净的工作目录下放好示例官方案例 xlsx
- [ ] 1080p 屏幕分辨率 + 关通知

### 录制中

- [ ] 屏幕录制：OBS Studio（免费）
- [ ] 屏幕录制要素：30fps · 5-10 Mbps bitrate
- [ ] 音频独立录制（不要边录边讲）
- [ ] 关键操作处放慢 0.5x，普通过场加速 2-3x

### 后期

- [ ] 字幕双语：中文 + 英文（LinkedIn 用）
- [ ] BGM 建议：YouTube Audio Library / Pixabay Music（注明 License）
- [ ] BGM 音量：-25dB（不抢戏）
- [ ] 数字加高亮框：3 小时 / 5 分钟 / 8 份交付件
- [ ] 输出多版本：15s.mp4 / 30s.mp4 / 60s.mp4 / GIF（关键演示片段）

## 七、转化漏斗预估

| 阶段 | 转化率 | 备注 |
|---|---|---|
| 视频曝光 → 完整观看 | 30-40% | 钩子做好的话 |
| 完整观看 → 点击 GitHub | 5-8% | 行业用户兴趣高 |
| GitHub 访问 → Star | 10-15% | 文档好的话 |
| GitHub 访问 → 实际安装 | 3-5% | 真实想用的咨询师 |
| 实际安装 → 提交反馈 / PR | 1-2% | 长期用户 |

**首发预期目标**（30 天）：
- 视频累计曝光 5-10 万
- GitHub Star 100-300
- 实际安装 30-100
- 提交 Issue/PR ≥5 个
