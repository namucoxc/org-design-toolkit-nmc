# 化工 · 子行业事项库

适用范围：精细化工 / 涂料 / 化学试剂 / 日化 / 中间体 等。
典型企业规模：50-500 人，年营收 5000 万 - 10 亿。

---

## 一、子行业特征

| 维度 | 化工行业 | 与通用制造业的差异 |
|---|---|---|
| 订单特征 | 大批量 + 长周期 | 通用制造业批次更小 |
| 客户结构 | B2B 工业客户 / 经销商 | 通用制造业含 B2C |
| 监管特征 | **强监管**（环评 / 安评 / 危化品证） | 通用制造业弱 |
| 工艺特征 | **连续化生产** + 批次化生产并存 | 通用制造业以离散为主 |
| 安全特征 | **重大安全风险**（爆炸 / 中毒 / 火灾） | 通用制造业风险低 |
| 物流特征 | **危化品物流**（专车 / 资质） | 通用制造业普通物流 |
| 财务特征 | 应收账款 60-120 天 | 类似 |

| # | 节点 | 行业标准时长 | 责任部门组合 |

| 7 | 订单评审 | 1-3 天 | 跟单 + 生产 + 采购 + 品管 |
| 8 | 原料采购 | 7-15 天 | 采购 |

## 三、推荐组织架构（10 个部门）

| 部门 | 编制建议 | 关键岗位 |
|---|---|---|
| 总经办 | 2-3 人 | 总经理 |
| 业务部 | 8-15 人 | 业务总监 / 大区经理 / 业务员 |
| 研发部 / 技术部 ⭐核心 | 8-20 人 | 研发总监 / 化学工程师 / 应用工程师 |
| 生产部 | 50-200 人 | 生产总监 / 车间主任 / 班组长 |
| 采购部 | 3-6 人 | 采购总监 / 采购员 |
| 品管部 / 化验室 ⭐独立 | 5-12 人 | 品管主管 / 化验员 / QA / QC |
| 仓储物流部 | 5-10 人 | 仓储主管 / 危化品仓管 |
| **EHS 部** ⭐⭐ 强制 | 3-8 人 | EHS 经理 / 安全员 / 环保专员 |
| 财务部 | 4-8 人 | 财务总监 / 财务主管 |
| 人力行政部 | 5-10 人 | 人力行政总监 |

### 关键设计原则

1. **EHS 部独立**——化工企业**必须**设独立 EHS 部门，直接向总经理汇报，敢于在生产/物流上"叫停"
2. **化验室独立于品管**——化工品质需化学分析能力，建议化验室单独建制
3. **研发部含应用工程师**——化工是技术驱动型，研发必须含"客户应用工程师"角色

## 四、推荐 KPI 范式

| 岗位 | 核心 KPI 1 | 核心 KPI 2 | 核心 KPI 3 |
|---|---|---|---|
| 总经理 | 年度营收 / 净利润率 | 一级目标达成率 | **重大安全事故 = 0** |
| 业务总监 | 营收完成率 | 应用方案转化率 | 新客户营收占比 |
| 研发总监 | 新品上市数 | 应用方案数量 | 客户技术满意度 |
| 生产总监 | 交期达成率 ≥95% | OEE | 单耗下降率 |
| **EHS 经理** ⭐ | **重大事故 = 0** | 三同时合规率 | 应急演练完成率 |
| 品管 / 化验主管 | 一次合格率 ≥98% | 化验时效 | 客户退货率 ≤0.5% |
| 采购总监 | 采购成本下降率 | 来料合格率 | 关键原料安全库存 |
| 财务总监 | DSO ≤90 天 | 现金流安全天数 | 预算偏差 ≤±10% |

## 五、典型陷阱

1. **EHS 隶属生产 → 监督失效** — EHS 必须独立，否则"自己监督自己"
2. **化验数据延迟出报告** — 化验滞后 ≥3 天会拖延整个交期
3. **危化品物流资质过期** — 必须建立物流商资质季度复审机制
4. **研发 → 量产工艺断层** — 必须设"中试车间"或"工艺工程师"承接
5. **客户技术服务弱** — 化工产品需现场培训和应用指导，否则客户流失快
6. **环保治理一次性投入大** — 必须 5 年规划，不能临时应付

## 六、推荐分析诊断重点

| 诊断点 | 关键问法 |
|---|---|
| EHS 成熟度 | 是否有独立 EHS 部门？过去 3 年重大事故起数？三同时合规情况？ |
| 研发驱动度 | 研发部人数占总人数比例？年度新品上市数？应用工程师配比？ |
| 化验室能力 | 化验设备投入？化验时效？是否第三方化验 vs 自建？ |
| 危化品物流 | 物流商资质审核机制？专车专人比例？应急预案完整度？ |
| 客户技术服务 | 客户工程师驻场比例？技术培训次数 / 年？ |
