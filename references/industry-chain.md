# 零售连锁 · 子行业事项库

适用范围：餐饮连锁 / 零售门店连锁 / 服务连锁（理发 / 美容 / 教培）等。
典型企业规模：30-500 家门店，员工 200-3000 人，年营收 1-20 亿。

---

## 一、子行业特征

| 维度 | 零售连锁 | 与制造业的差异 |
|---|---|---|
| 业务模式 | **多门店 + 总部赋能** | 制造业是单工厂或多工厂 |
| 收入来源 | 终端门店日营业额 | 制造业是订单收入 |
| 客户结构 | C 端散客 + 会员 | 制造业是 B 端客户 |
| 扩张关键 | **可复制 SOP + 单店模型** | 制造业是产能扩张 |
| 监管特征 | 食品 / 经营许可 / 消防 / 商标 | 类似 |
| 财务特征 | 现金流好 / 应收账款少 | 制造业 DSO 长 |
| 周期特征 | 强季节 + 强地段 | 类似但弱 |

## 三、推荐组织架构（10 个部门 + 门店）

| 部门 | 编制建议 | 关键岗位 |
|---|---|---|
| 总经办 | 2-3 人 | 总经理 / 董事长 |
| 品牌市场部 ⭐核心 | 5-15 人 | 品牌总监 / 市场经理 / 私域 / 公关 |
| **拓展部**（选址 / 加盟）⭐核心 | 5-15 人 | 拓展总监 / 选址专员 / 加盟商管理 |
| **工程部**（装修 / 维保）| 3-10 人 | 工程主管 / 装修项目经理 |
| **营运部**（督导 / 运营标准）⭐核心 | 10-30 人 | 营运总监 / 营运督导 / 大区经理 |
| 研发部（新品 / SOP）| 5-15 人 | 研发总监 / 新品经理 / SOP 经理 |
| 供应链部（采购+仓储+配送）| 10-30 人 | 供应链总监 / 采购 / 仓储 / 配送 |
| 品控部 ⭐独立 | 3-10 人 | 品控主管 / 神秘客 |
| 会员部 / 私域部 | 3-8 人 | 会员运营 / 数据分析 |
| 财务+人力行政+法务 | 10-25 人 | 财务总监 / HRD / 法务 |
| **门店** | N × 单店编制 | 店长 / 副店长 / 店员 / 收银 |

### 关键设计原则

1. **拓展、营运、品牌是连锁三大引擎**——拓展决定增量，营运决定存量，品牌决定单价
2. **营运督导制是连锁标配**——大区督导 / 区域督导 / 店长三级，必须建立
3. **加盟模式额外加 "加盟商支持部"**——分离直营/加盟两条线
4. **数据驱动是核心能力**——必须设独立会员/数据部，否则连锁等于"开很多店却各自为政"

## 四、推荐 KPI 范式

| 岗位 | 核心 KPI 1 | 核心 KPI 2 | 核心 KPI 3 |
|---|---|---|---|
| 总经理 | 年度营收 / 净利润率 | 新开门店数 | 同店增长率 |
| 拓展总监 | 新开店达成率 | **单店盈亏平衡周期** ≤6 月 | 选址成功率 |
| 营运总监 | 同店增长率 | 单店净利率 | 督导覆盖率 |
| 品牌市场总监 | 品牌曝光 / 客单价 | 会员增长 | 营销 ROI |

| 品控主管 | 神秘客评分 | 食安/服务投诉率 | 门店巡检合格率 |
| 财务总监 | 现金流安全天数 | DSO | 单店财务可视化 |
| 店长 | 单店营业额完成率 | 单店净利率 | 客户满意度 |

## 五、典型陷阱

1. **单店模型未跑通就规模扩张** — 必须先用 3-5 家直营店跑通"单店盈亏平衡周期 ≤6 月"
2. **总部臃肿 / 门店薄弱** — 总部成本占比应 ≤15%
3. **督导监督失效** — 督导制是连锁命门，必须考核到位
4. **SOP 仅停留在纸面** — SOP 必须有视频化 + 考核 + 巡检三位一体
5. **加盟商管理混乱** — 加盟商有独立利益主体，必须建标准化合同 + 培训 + 督导 + 退出机制
6. **门店数据黑箱** — 必须实时数据驱动（POS / 会员 / 库存全打通）
7. **新品上市慢** — 连锁要求新品上市周期 ≤45 天，不能像制造业 3-6 个月

## 六、推荐分析诊断重点

| 诊断点 | 关键问法 |
|---|---|
| 单店模型成熟度 | 单店盈亏平衡周期？同店增长率？标杆店利润率？ |
| 督导体系 | 督导人数 / 门店数比例？督导巡检频次？督导考核机制？ |
| SOP 成熟度 | 是否有 SOP 视频？是否进行 SOP 考核？SOP 与实际操作差异率？ |
| 加盟 vs 直营 | 加盟门店占比？加盟商满意度？加盟商退出率？ |
| 数据成熟度 | 是否 POS / 会员 / 库存数据实时打通？数据驱动决策的占比？ |
| 新品节奏 | 年度新品上线数？新品贡献度？新品上市周期？ |
