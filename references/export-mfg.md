# 外贸出口制造业 · 子行业事项库

适用范围：以出口为主要业务的制造企业（出口占比 ≥30%），覆盖电子 / 家具 / 服装 / 工具 / 五金等。
典型企业规模：100-1000 人，年营收 5000 万 - 10 亿。

---

## 一、子行业特征

| 维度 | 外贸出口 | 与国内制造业的差异 |
|---|---|---|
| 客户结构 | 海外采购商 / 海外品牌方 / 海外经销商 | 国内偏 B 端工业客户 |
| 订单特征 | 小批量 + 多批次 + 长周期（含海运）| 国内更短周期 |
| 监管特征 | **进出口资质** / **目标国法规**（FDA/CE/RoHS 等）| 国内监管不同 |
| 物流特征 | **海运 + 陆运 + 清关** | 国内单一物流 |
| 财务特征 | **外汇结算** + **出口退税** + DSO 90-180 天 | 国内更短 |
| 风险特征 | **地缘政治** / **汇率** / **客户信用** | 国内风险不同 |

## 三、推荐组织架构（10 个部门）

| 部门 | 编制建议 | 关键岗位 |
|---|---|---|
| 总经办 | 2-3 人 | 总经理 |
| 业务部 | 10-25 人 | 业务总监 / 大区经理 / 业务员（**多语种**）|
| **跟单部** ⭐隐形枢纽 | 5-10 人 | 跟单主管 / 跟单专员 / **清关专员** / **退税专员** |
| 研发部 / 技术部 | 5-15 人 | 研发总监 / 工程师 |
| 生产部 | 50-300 人 | 生产总监 / 车间主任 |
| 采购部 | 3-10 人 | 采购总监 / 采购员 |
| 品管部 | 5-15 人 | 品管主管 / IPQC / OQC |
| 仓储 + 物流部 | 5-10 人 | 物流主管 / **报关员** |
| 财务部 ⭐含外汇专员 | 5-10 人 | 财务总监 / **外汇 / 退税专员** |
| 人力行政部 | 5-10 人 | HRD |

### 关键设计原则

1. **跟单部独立**——外贸订单流程长达 90-180 天，跟单是核心枢纽
2. **业务员必须多语种**——至少英语流利，部分小语种（西语 / 阿语 / 法语 / 日语）
3. **财务设外汇 + 退税专员**——外汇敞口和退税是利润大头
4. **物流部独立于仓储**——报关 + 海运是专业能力，必须设专人

## 四、推荐 KPI 范式

| 岗位 | 核心 KPI 1 | 核心 KPI 2 | 核心 KPI 3 |
|---|---|---|---|
| 总经理 | 年度营收 / 净利润率 | **汇率风险敞口控制** | 一级目标达成率 |
| 业务总监 | 营收完成率 | 新客户开发数 | 客户信用控制 |
| 跟单主管 | 全流程时长（目标 ≤120 天）| 清关一次通过率 | 客诉响应及时率 |
| 研发总监 | 新品上市数 | 客户技术满意度 | 各国认证完成率 |
| 生产总监 | 交期达成率 ≥95% | 一次合格率 ≥98% | OEE |
| 财务总监 | **DSO ≤120 天** | **出口退税到账周期 ≤90 天** | **外汇套保比例** |
| **报关员 / 清关专员** | 报关一次通过率 ≥98% | 清关延误事故 = 0 | 进口商资质合规 |
| 品管主管 | 一次合格率 | 海外验货通过率 | 客诉关闭时效 |

## 五、典型陷阱

1. **汇率波动吞噬利润** — 必须建立外汇套保机制（远期结售汇 / 锁汇）
2. **客户信用风险** — 必须用中信保等机构做信用评估，不接 D/A 单
3. **目标国法规更新** — CE / FDA / RoHS / FCC 等更新必须有专人盯
4. **出口退税延期** — 退税专员能力差 → 资金占用大
5. **海运延期 / 港口拥堵** — 必须有"延期应对预案" + 与多家船公司合作
6. **客户验货严格** — 必须有第三方验货（SGS / 天祥）配合机制
7. **跨时区沟通** — 业务员必须有"接力工作制"（白班 + 夜班覆盖客户工作时间）

## 六、推荐分析诊断重点

| 诊断点 | 关键问法 |
|---|---|
| 出口占比 | 出口营收占比？目的国分布？地缘风险评估？ |
| 跟单成熟度 | 全流程平均时长？跟单 SOP 完整度？清关一次通过率？ |
| 财务外贸专业度 | 外汇敞口管理？退税到账周期？是否有外汇套保？ |
| 客户信用 | 是否用信保？应收账款坏账率？大客户信用评估机制？ |
| 各国认证 | 主要目标国法规掌握度？认证 / 备案完整度？ |
| 海运合作 | 合作船公司数量？港口延误应对预案？ |
