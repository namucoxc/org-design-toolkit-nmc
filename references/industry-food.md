# 食品 · 子行业事项库

适用范围：食品加工 / 饮料 / 烘焙 / 调味品 / 速冻食品 等。
典型企业规模：50-1000 人，年营收 5000 万 - 10 亿。

---

## 一、子行业特征

| 维度 | 食品行业 | 与通用制造业的差异 |
|---|---|---|
| 订单特征 | 大批量 + 季节性 + 短保质期 | 通用制造业无保质期约束 |
| 客户结构 | 经销商 / KA 大客户 / 电商 | 通用制造业偏纯 B2B |
| 监管特征 | **强监管**（食品生产许可证 / HACCP / SC 认证 / FDA 等） | 通用制造业弱 |
| 工艺特征 | 流水线 + 严格卫生控制 | 通用制造业卫生要求低 |
| 财务特征 | DSO 30-60 天（KA 客户长） | 类似 |
| 周期特征 | **季节性强**（节假日 / 旺淡季） | 通用制造业波动小 |

## 二、14 节点创利流程（食品专用）

| # | 节点 | 行业标准时长 | 责任部门组合 |
|---|---|---|---|
| 1 | 市场调研与新品创意 | 30-60 天 | 市场 + 研发 |
| 2 | 配方研发与试制 | 30-90 天 | 研发 |
| 3 | 法规与备案 | 60-180 天 | 法规事务 + 研发 |
| 4 | 客户验证与确认 | 30-60 天 | 业务 + 客户 |
| 5 | 包装设计 | 14-30 天 | 包装设计 / 市场 |
| 6 | 合同签约 | 1-3 天 | 业务 + 法务 + 财务 |
| 7 | **HACCP / 食安评审** | 持续 | 品控 / 食安 |
| 8 | 订单评审 | 1-2 天 | 跟单 + 生产 + 采购 |
| 9 | 原料采购（**含食品级认证**） | 5-15 天 | 采购 |
| 10 | 生产排程 | 1-2 天 | 生产计划 |
| 11 | 生产制造（**含洁净要求**） | 1-7 天 | 生产 |
| 12 | 质量检验（**含微生物 / 理化**） | 2-7 天 | 品控 / 化验 |
| 13 | 包装入库（**含批号 / 保质期**） | 1-2 天 | 仓储 |
| 14 | 物流配送（**含冷链 / 时效**） | 1-7 天 | 物流 + 业务 |

## 三、推荐组织架构（10 个部门）

| 部门 | 编制建议 | 关键岗位 |
|---|---|---|
| 总经办 | 2-3 人 | 总经理 |
| 市场部 ⭐独立 | 5-15 人 | 市场总监 / 品牌经理 / 新品规划 |
| 业务部 | 8-20 人 | 业务总监 / KA 经理 / 经销商管理 |
| 研发部 ⭐核心 | 5-15 人 | 研发总监 / 食品工程师 / 风味师 |
| **法规事务部** ⭐强制 | 2-5 人 | 法规专员 / 注册专员 |
| 生产部 | 50-300 人 | 生产总监 / 车间主任 |
| 采购部 | 3-8 人 | 采购总监 |
| **品控部 / 食安部** ⭐强制 | 5-15 人 | 品控主管 / HACCP 经理 / 化验员 |
| 物流仓储部 | 5-12 人 | 物流主管 / 冷链调度 |
| 财务 + 人力行政 | 8-15 人 | 财务总监 / 人力行政总监 |

### 关键设计原则

1. **市场部独立于业务部**——食品行业品牌驱动强，市场必须独立做新品规划
2. **法规事务部独立**——食安法规复杂，必须有专人盯（产品注册 / 标签合规 / 出口备案）
3. **品控/食安部直接向总经理汇报**——食品安全一票否决，必须独立监督

## 四、推荐 KPI 范式

| 岗位 | 核心 KPI 1 | 核心 KPI 2 | 核心 KPI 3 |
|---|---|---|---|
| 总经理 | 年度营收 / 净利润率 | **食品安全事故 = 0** | 一级目标达成率 |
| 市场总监 | 新品上市数 | 品牌曝光 / 转化 | 新品营收占比 |
| 业务总监 | 营收完成率 | 经销商满意度 | KA 客户营收占比 |
| 研发总监 | 新品研发周期 | 配方稳定性 | 客户接受度 |
| **食安/品控主管** ⭐ | **食安事故 = 0** | 一次合格率 ≥99% | HACCP 合规率 |
| 法规专员 | 注册成功率 | 合规审计零违规 | 出口备案及时率 |
| 生产总监 | 交期达成率 | OEE | 单位产品成本 |
| 采购总监 | 食品级原料合格率 | 采购成本下降率 | 关键原料备库 |

## 五、典型陷阱

1. **品控隶属生产** — 必须独立，否则品质让位于产量
2. **法规事务依赖外包** — 关键法规必须内化能力
3. **新品上市无法规预审** — 上市后被监管叫停，损失惨重
4. **冷链断链** — 物流商资质必须季度复审，温度监控全程闭环
5. **季节性产能不足** — 旺季外协标准必须提前 3 个月规划
6. **经销商窜货** — 必须建窜货识别 + 处罚机制
7. **配方泄密** — 研发部必须建保密协议 + 关键配方权限分级

## 六、推荐分析诊断重点

| 诊断点 | 关键问法 |
|---|---|
| 食安成熟度 | 是否有 HACCP / ISO22000 / SQF 认证？过去 3 年食安事故起数？ |
| 法规事务能力 | 是否有专职法规岗？新品上市平均备案周期？ |
| 新品研发节奏 | 年度新品上市数？新品营收占比？新品成功率？ |
| 渠道结构 | 经销商 / KA / 电商占比？大客户依赖度？ |
| 冷链管理 | 是否需要冷链？冷链断链率？物流商资质审核机制？ |
| 配方与品牌资产 | 是否有专利？商标布局？配方泄密风险防控？ |
