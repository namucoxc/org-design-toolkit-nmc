# 多地协同 · 叠加层模板

> 本模板**不单独使用**，而是与任一行业模板（精品包装/通用制造业/化工/食品/零售连锁/项目型）叠加使用。
>
> 适用：客户有"总部 + 工厂"、"多地工厂"、"多地办公"、"集团 + 子公司"等跨地结构。

---

## 一、多地协同典型场景

| 场景 | 典型架构 | 举例 |
|---|---|---|
| **总部 + 单工厂** | 业务/管理在城市，生产在产业带 | 精品包装：广州办公 + 江西工厂 |
| **总部 + 多工厂** | 总部统辖多个工厂 | 化工：上海总部 + 浙江 / 山东 / 广东多工厂 |
| **多地办公** | 多个城市设分公司 | 项目型：北京 / 上海 / 深圳三地 |
| **集团 + 子公司** | 集团控股多个独立法人 | 集团：母公司 + 多事业部子公司 |
| **直营 + 加盟** | 总部直营基地 + 加盟门店 | 零售连锁：总部 + N 家加盟 |

---

## 二、多地协同要解决的 5 大冲突

| # | 冲突 | 表现 | 叠加层动作 |
|---|---|---|---|
| 1 | **权责不清** | 总部下决策但工厂不执行 / 工厂自作主张 | 矩阵增加"总部 vs 工厂"主责清单 |
| 2 | **信息不通** | 总部不知道工厂实情 / 工厂不知道总部目标 | 设"工厂联络员" + 月度联席会 |
| 3 | **流程脱节** | 订单从总部到工厂卡点多 | 矩阵增"跨地协同"一级模块 |
| 4 | **文化分裂** | 总部 vs 工厂员工各自一套 | 高管轮岗 + 联合培训 |
| 5 | **财务核算混乱** | 总部 vs 工厂账务无法对齐 | 集中财务 + 工厂会计 BP |

---

## 三、叠加在矩阵的 5 个一级模块

在原行业模板的基础上，**新增以下 5 个一级流程模块**：

| 模块 | 关键事项示例 |
|---|---|
| **跨地协同·决策** | 战略目标分解到各地 / 重大投资审批 / 高管定岗 |
| **跨地协同·运营** | 月度运营联席会 / 跨地订单流转 / 工厂排产对接 |
| **跨地协同·信息系统** | ERP / CRM / OA 跨地打通 / 数据一体化 |
| **跨地协同·人才** | 跨地干部轮岗 / 联合培训 / 异地任职津贴 |
| **跨地协同·文化** | 总部 vs 工厂联合活动 / 价值观对齐 |

## 四、推荐新增岗位

| 岗位 | 编制 | 职责 |
|---|---|---|
| **工厂联络员** ⭐ | 每工厂 1 人 | 跨地信息对接、问题升级、月度报表 |
| **总部驻厂代表** | 视规模 1-3 人 | 大型工厂可派"总经理派驻代表" |
| **集团运营官 / COO** | 1 人 | 统辖跨地运营 |
| **集团 IT 总监** | 1 人 | 跨地信息系统统一规划 |
| **集团财务 BP** | 每工厂 1-2 人 | 工厂财务直接对接集团 |

---

## 五、推荐 KPI 范式（叠加层）

| 岗位 | 多地协同 KPI |
|---|---|
| 总经理 | 跨地月度联席会出席率 = 100% / 总部派驻规模占比 |
| **工厂联络员** ⭐ | 信息上报时效 ≤24h / 跨地问题升级数 |
| 集团运营官 | 跨地订单流转时效 / 跨地协同 NPS |
| 工厂厂长 | 工厂 vs 总部目标达成对齐度 / 工厂自决策事项占比 |
| 集团 IT 总监 | 跨地数据一致性 ≥99% / 系统打通完成率 |
| 集团财务 BP | 工厂月报合规率 / 跨地核算误差率 ≤1% |

## 六、典型陷阱

1. **总部下指令而无执行抓手** — 必须有"工厂联络员"承接，否则指令断头
2. **工厂强人独立王国** — 工厂厂长有 80% 决策权时，集团一致性失控
3. **跨地财务无法日清** — ERP 必须打通到工厂，月底对账延期 ≥7 天就出问题
4. **跨地差旅成本失控** — 必须有差旅政策 + 异地任职津贴
5. **总部信息黑箱** — 总部决策依赖工厂上报的滞后数据，决策必然失真
6. **文化两套并行** — 总部"快慢节奏"与工厂"传统节奏"必须做文化对齐

## 七、推荐分析诊断重点

| 诊断点 | 关键问法 |
|---|---|
| 多地权责清晰度 | 总部 vs 工厂的决策清单是否成文？过去 1 年权责争议事件多少起？ |
| 信息系统打通度 | ERP / CRM 是否跨地打通？跨地数据一致性多少？ |
| 跨地联席会节奏 | 月度 / 季度联席会是否常态化？纪要落实率？ |
| 干部轮岗 | 总部 vs 工厂高管是否互轮？轮岗周期？ |
| 跨地财务一致性 | 月报合并周期？跨地核算误差？工厂财务上报合规率？ |
