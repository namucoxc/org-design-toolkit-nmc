# 精品包装制造 · 子行业事项库（references/precious-packaging.md）

适用范围：礼盒包装 / 项链盒 / 香水盒 / 首饰盒 / 烟酒盒 / 月饼盒 / 化妆品盒 等中高端定制化包装。
典型企业规模：年营收 5000 万 - 3 亿，员工 100 - 500 人，B2B 出口为主。

---

## 一、子行业特征

| 维度 | 精品包装特征 | 与通用制造业的差异 |
|---|---|---|
| 订单特征 | 小订单（5K-50K 件） / 多批量 / 高定制化 | 通用制造业偏向大批量 |
| 客户结构 | 客户集中度高（单一客户占比 20-30%） | 通用制造业较分散 |
| 出口结构 | 出口业务超 50%（中东 / 欧美为主） | 通用制造业以国内为主 |
| 工艺特征 | 印刷 + 裱糊 + 手工 + 烫金 + 包装 五工段 | 通用制造业多以注塑/冲压等为主 |
| 设计依赖 | 设计能力是核心竞争力（往往依赖外协设计师） | 通用制造业设计弱 |
| 交付周期 | 100-130 天（含打样 + 货期 + 出货） | 通用制造业 30-60 天 |
| 财务特征 | 外贸退税 + 应收账款 DSO 长 | 通用制造业 DSO 短 |

## 二、16 节点核心创利流程（精品包装专用）

| # | 节点 | 行业标准时长 | 责任部门组合 |
|---|---|---|---|
| 1 | 打样 | 14-29 天 | 研发部（含样板/技术）+ 采购 + 业务 + 跟单 |
| 2 | 货期评估与承诺 | 7-15 天 | 业务 + 生产 + 跟单 + 采购 + 资源 |
| 3 | 合同与定金 | 1-3 天 | 财务 + 业务 + 跟单 |
| 4 | 订单评审会 | 3-6 天 | 跟单 + 生产 + 技术 + 采购 + 仓储 + 品管 |
| 5 | 采购物料 | 3-6 天 | 采购 + 业务 + 跟单 |
| 6 | 刀模设计与制作 | 2-4 天 | 研发（技术工程师）+ 业务 + 跟单 |
| 7 | 烫金工艺 | 1-2 天 | 生产（烫金工段）+ 业务 + 跟单 |
| 8 | 产前样制作与确认 | 3-6 天 | 生产 + 品管 + 业务 + 跟单 |
| 9 | 上线生产与大货样 | 1-3 天 | 品管 + 生产 + 业务 + 跟单 |
| 10 | 装箱单 | 1-5 天 | 生产 + 仓储 |
| 11 | 仓库入库与备货 | 1-2 天 | 业务 + 跟单 + 仓储 |
| 12 | 出货一·港口装柜 | 14-22 天 | 业务 + 跟单 + 生产 + 仓储 |
| 13 | 出货二·目的港交付 | 7-11 天 | 业务 + 跟单 |
| 14 | 清关资料 | 1-2 天 | 业务 + 跟单 |
| 15 | 在途跟进 | 1-2 天 | 业务 + 跟单 |
| 16 | 出口退税 | 3-6 天 | 财务 + 业务 |

**总时长**：60-124 天 / 单订单（行业平均 ~100 天）
**典型瓶颈**：打样（占总时长 25-30%）+ 出货一二（占 25-30%）

## 三、推荐组织架构（10 个部门）

| 部门 | 编制建议（中型企业 200-300 人） | 关键岗位 |
|---|---|---|
| 总经办 | 2-3 人 | 总经理 |
| 业务部 | 15-25 人 | 业务总监 / 大区经理 / 大客户经理 / 业务员 |
| 跟单部 ⭐隐形枢纽 | 5-8 人 | 跟单主管 / 跟单组长 / 清关专员 / 退税专员 |
| 研发部（设计+样板+技术整合）| 10-15 人 | 研发总监 / 样板主管 / 技术工程师 / 设计师 / 打样工程师 |
| 生产部 | 130-180 人 | 生产总监 / 工厂联络员 / 计划主管 / 五工段长（印刷/裱糊/手工/包装/烫金）|
| 采购部（含资源/外协） | 4-7 人 | 采购总监 / 采购经理 / 物控 |
| 品管部 | 6-10 人 | 品管主管 / IPQC / OQC |
| 仓储部（独立于生产）| 4-8 人 | 仓储主管 |
| 财务部 | 5-8 人 | 财务总监 / 财务主管 / 出纳 / 税务/退税专员 |
| 人力行政部 | 6-10 人 | 人力行政总监 / 招聘培训主管 |

### 关键设计原则

1. **跟单部独立**——在 16 节点中跟单部出现 ≥12 次，是研产销协同的隐形枢纽
2. **研发部整合**——设计 + 样板（打样）+ 技术（刀模）三者必须合一，否则打样周期无法压缩
3. **仓储部独立**——出货一二占总时长 25-30%，仓储独立才能聚焦
4. **品管部独立于生产**——敢于停线 / 出货放行 / 客诉 8D 独立调查

## 四、推荐 KPI 范式

| 岗位 | 核心 KPI 1 | 核心 KPI 2 | 核心 KPI 3 |
|---|---|---|---|
| 总经理 | 年度营收 / 净利润率 | 一级目标达成率 | 现金流安全天数 |
| 业务总监 | 营收完成率 | 出业绩业务员占比 | 新客户营收占比 |
| 跟单主管 ⭐ | 全流程时长（目标 ≤90 天） | 订单评审准时率 ≥95% | 交期承诺达成率 ≥95% |
| 研发总监 ⭐ | 打样响应时效（目标 ≤14 天） | 设计稿一次通过率 ≥70% | 外协转内部签约率 |
| 生产总监 | 交期达成率 ≥95% | 一次性合格率 ≥98% | 人均产值 |
| 财务总监 | 现金流安全天数 ≥90 | DSO ≤75 天 | 出口退税到账周期 ≤90 天 |
| 品管主管 | 一次性合格率 ≥98% | 客户退货率 ≤0.5% | 客诉 8D 关闭 ≤30 天 |
| 样板主管 | 打样响应时效 ≤14 天 | 一次通过率 ≥70% | 打样转量产率 ≥40% |

## 五、典型陷阱

1. **业务员出业绩率低**——精品包装业务员"靠老板带"，新人无法独立成单；必须建提成机制 + 师徒带教
2. **打样周期过长**——研发能力分散在样板/技术/设计 3 个独立部门导致；整合为研发部能大幅压缩
3. **单一市场依赖**——50% 以上营收来自单一海外市场，地缘政治风险大；必须开发第二增长曲线
4. **外协设计师风险**——设计师纯外协合作，客户定制响应慢；战略上转内部岗
5. **双工厂协同**——总部在城市、工厂在产业带，跨地协同必须设专人（工厂联络员）
6. **应收账款 DSO 长**——出口业务 DSO 通常 90-120 天；财务+业务联合催收
7. **手工品质风险**——精品包装大量人工工序，品质不稳定；必须独立品管部 + 全检 + 大货样

## 六、推荐分析诊断重点

咨询师做精品包装企业的方案设计时，特别注意：

| 诊断点 | 关键问法 |
|---|---|
| 流程瓶颈 | 客户实际打样周期多少天？出货周期多少天？瓶颈节点占总时长比例？ |
| 跟单成熟度 | 现有跟单是否独立部门？跟单人员归属哪？是否有跟单 SOP？ |
| 研发整合度 | 设计/样板/技术目前是 1 个部门还是 3 个？外协设计师占比多少？ |
| 双工厂协同度 | 总部和工厂如何沟通？多久开一次联席会？谁是工厂联络员？ |
| 客户集中度 | TOP3 客户占营收比例？最大单一客户占比？市场分布？ |
| 财务管控成熟度 | DSO 天数？外汇风险敞口？出口退税到账周期？是否有月度经营分析会？ |
