# 制造业-通用 · 子行业事项库

适用范围：标准化制造业（机械加工 / 五金 / 注塑 / 电子组装等）。
典型企业规模：100-1000 人，年营收 5000 万 - 5 亿。

---

## 一、子行业特征

| 维度 | 通用制造业 |
|---|---|
| 订单特征 | 大批量 / 中等定制化 |
| 客户结构 | 中等集中（TOP10 占 50-60%） |
| 业务模式 | 国内 + 出口（约 50-50） |
| 工艺特征 | 标准化工序 + 部分定制工序 |
| 交付周期 | 30-60 天 |
| 财务特征 | DSO 60-90 天 |

## 四、推荐 KPI 范式

| 岗位 | 核心 KPI 1 | 核心 KPI 2 | 核心 KPI 3 |
|---|---|---|---|
| 总经理 | 年度营收 / 净利润率 | 一级目标达成率 | 现金流安全天数 |
| 业务总监 | 营收完成率 | 新客户开发数 | 应收账款回款率 |
| 生产总监 | 交期达成率 ≥95% | 一次合格率 ≥98% | OEE ≥75% |
| 技术总监 | 新品研发周期 | 工艺改进项数 | 客户技术满意度 |
| 采购总监 | 采购成本下降率 ≥3% | 按时交付率 ≥95% | 呆滞库存 ≤5% |
| 品管主管 | 一次合格率 ≥98% | 客户退货率 ≤0.5% | 客诉关闭时效 |
| 财务总监 | DSO ≤75 天 | 月报时效 ≤10 天 | 预算偏差 ≤±10% |

## 五、典型陷阱

1. **生产计划与销售脱节** — 生产排单凭老板拍脑袋，无产能负荷分析
2. **采购降本难推进** — 单一供应商绑定多年，缺乏比价机制
3. **品质事故依赖救火** — 无 SOP / 无 8D / 客诉响应慢
4. **车间数据黑箱** — 无 ERP / 凭师傅经验，管理半径有限
5. **大客户依赖** — TOP1 客户占比超 30%，风险集中
